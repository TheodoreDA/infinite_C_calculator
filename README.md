## Getting Started
    
#### Languages and libraries used in this project (separate with commas):

#### Installation: 
```  
git clone git@github.com:TheodoreDA/infinite_C_calculator.git
```
```  
cd infinite_C_calculator && make
```

#### Usage:
```  
./bistro "6*7"
```
